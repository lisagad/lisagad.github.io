HOW DID YOU END UP ?

[Jeu de société] [2024] [Travail de groupe] [Scolaire][Illustration] [Procreate]

How did you end up est un jeu de société créé dans le cadre d'un workshop Game Design dirigé par Tatiana Vilela Dos Santos. En groupe, nous devions créer un jeu fonctionnel et cohérent, tout en respectant les mécaniques de jeu imposées. Sur ce projet, je m'occupais principalement de la partie illustration.

Avec la participation de : Vita Kryzhanivska, Lisa Godhino Da Silva, Houda El Asri, Manon Faroult, Nathan Macé, Nathan Sonzogni.

[✮ In english]

How Did You End Up is a board game created as part of a Game Design workshop led by Tatiana Vilela Dos Santos. In groups, we were tasked with creating a functional and cohesive game while adhering to imposed game mechanics. On this project, I was mainly responsible for the illustration part.

With the participation of: Vita Kryzhanivska, Lisa Godhino Da Silva, Houda El Asri, Manon Faroult, Nathan Macé, Nathan Sonzogni.
